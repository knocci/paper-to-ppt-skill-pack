# Slides Plan Example

## Presentation Meta

```yaml
paper_title: 待填论文标题
presenter: 待填
 audience: 研一组会
duration_minutes: 12
slide_count: 10
language: 简体中文
aspect_ratio: 16:9
style: 学术、简洁、清晰、少文字、多图示
source_pdf: sources/example/example.pdf
```

## Storyline

1. 这篇论文关注一个在实际应用中反复出现的问题。
2. 现有方法在某个关键场景下存在不足。
3. 作者提出一个核心洞察，并把它落到具体方法结构里。
4. 实验证据表明这个设计带来了主要收益。
5. 最后讨论它的适用边界和对后续工作的启发。
