# Review Checklist

## 1. PDF 读取完整性

- [ ] PDF 已完整读取
- [ ] 标题、作者、年份已核实
- [ ] 摘要、引言、方法、实验、结论均已覆盖
- [ ] 不可读页面已列出
- [ ] 不确定内容已标注“待核实”

## 2. 事实准确性

- [ ] 没有编造论文内容
- [ ] 没有编造实验结果
- [ ] 没有编造图表或公式
- [ ] 所有关键结论都有 source_reference
- [ ] 所有数字都有 source_reference
- [ ] 所有方法细节都有 source_reference

## 3. 汇报逻辑

- [ ] 页面规划不是按论文目录机械复述
- [ ] 故事线清楚：问题 → 缺口 → 方法 → 证据 → 启发
- [ ] 每页只有一个核心观点
- [ ] 标题是观点式标题
- [ ] 10-15 分钟可以讲完

## 4. 页面结构规划

- [ ] 页数为 10-12 页，或用户指定页数
- [ ] 规划为 16:9
- [ ] 少文字，多图示
- [ ] 每页 bullet 不超过 3 条
- [ ] 方法页规划了流程图、模块图或论文原方法图
- [ ] 实验页突出关键对比
- [ ] 结论页包含贡献、局限与启发

## 5. 图表与图片交接

- [ ] 关键图表已提取或已列入 images_needed.md
- [ ] 缺失图片已明确标注手动截图要求
- [ ] 多图多表页面已说明是否放大、对比、多面板、拆页或附录
- [ ] 大表格已说明建议裁剪的关键行列
- [ ] 图表来源已标注
- [ ] 没有使用虚构图片或虚构数据

## 6. 讲稿

- [ ] speaker_notes.md 已覆盖每一页
- [ ] 每页讲稿包含转场句
- [ ] 讲稿没有大段照搬论文原文
- [ ] 待核实内容在讲稿中已明确提示

## 7. 待用户处理

- [ ] 需要手动截图的图片已列出
- [ ] 需要核实的论文内容已列出
- [ ] 需要人工确认的图表裁剪已列出
