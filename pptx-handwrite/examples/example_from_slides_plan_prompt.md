# Example prompt: preview slides from handoff files

Use the pptx skill to generate only 1-2 preview slides from the current handoff directory.

Read:
- slides_plan.md
- images_needed.md
- speaker_notes.md
- paper_summary.md
- assets/

Do not regenerate the whole deck yet. Select one method slide and one experiment/result slide, then create `draft_preview.pptx` and thumbnails for review.

Use the default hand-drawn academic sketch style:
- English headings: Kalam
- English body: Patrick Hand
- Chinese fallback: FangSong / 仿宋
- Equations and paper symbols: Cambria Math or another math/scientific font
- No newly added text below 14 pt
- Do not force every figure/table into a card; maximize figure/table readability first
- Missing visuals should be placeholders, not fabricated images
